Відомі баги:
1) Інколи створюється два вікна замість одного ( причину ще не вияснив )
2) Один процес не вмирає, що тягне за собою те, що він не закриває за собою сокет.
3) Дані наразі ніяк не шифруються.
4) Сервер/клієнт/тестер не мають таймінгів та містять подвійне зчитування. Це може стопнути сервер/клієнт/тестер.
5) ui - виключно тому що забув їх прибрати. Вони не несуть ніякої інформації.
6) Спілкування client -> test_receiver та test_sender -> client відбувається корректно в більшості випадків, тим не меннш client <-> client працювати відмовляється.
7) DNS сервер часто зависає ( не зробив перевірку чи вдалось підключитись до сокета, внаслідок чого візуально можна це помітити лише коли він не видає жодної реакції на запити ).
8) DNS не брацює з БД, лише з файловою системою. Можна його без труднощів доробити. Також наразі не передбачено видалення та перевірка на повтори. Не зробив ще оскільки не був впевнений чи варто.
